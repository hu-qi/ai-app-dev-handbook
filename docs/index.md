## 这不是复刻，而是原创对标

本手册参考优秀 AI 学习产品的体验：清晰目录、阶段学习、项目驱动、部署友好。但所有教程、案例、代码与表达均为原创。

## 推荐学习顺序

1. 先理解 AI 应用开发全景图
2. 再掌握大模型 API 与 Prompt 工程
3. 然后进入 RAG、工具调用和 Agent
4. 最后完成四个项目，并学习部署与商业化
