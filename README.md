# AI 应用开发实战手册

原创系列教程：从 Prompt 到 Agent 产品化。

## 项目定位

本项目对标优秀 AI 学习手册的产品形态，但内容、章节表达、代码案例与项目设计均为原创。

面向开发者、产品经理、AI 创业者，系统覆盖：

- AI 应用开发认知
- 大模型 API 调用
- Prompt 工程
- RAG 知识库
- Function Calling / Tool Calling
- Agent 工作流
- 国产大模型接入
- 工程化部署
- AI SaaS 产品化
- 求职与商业化

## 本地运行

```bash
npm install
npm run dev
```

## Vercel 部署

- Framework Preset: VitePress
- Build Command: `npm run build`
- Output Directory: `docs/.vitepress/dist`
- Install Command: `npm install`
